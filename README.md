# Entelligo Pvt Website

A responsive company website built using Next.js and Tailwind CSS.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Home, About, and Contact pages
- Responsive navbar with active page highlighting
- Mobile navigation menu
- Transparent navbar with scroll effect
- Interactive hover animations
- Modern UI design using Tailwind CSS
- Contact form with client-side functionality

## Technologies Used

- Next.js
- React.js
- Tailwind CSS
- JavaScript

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/GauRaVsinghbora/Entelligo_pvt.git
```

Go into the project directory:

```bash
cd Entelligo_pvt
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open browser:

http://localhost:3000

## Build for Production

```bash
npm run build
npm start
```
